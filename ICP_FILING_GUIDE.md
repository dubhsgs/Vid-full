## 中国 ICP 备案完全指南

### 📋 备案基础知识

#### 什么是 ICP 备案？
ICP（Internet Content Provider）备案是中国大陆对网站的强制性许可制度。任何在中国大陆境内提供互联网信息服务的网站都必须进行备案。

#### 备案类型
1. **ICP 备案**：非经营性网站（免费服务）
2. **ICP 许可证**：经营性网站（收费服务、广告收入等）

**vaid.top 当前状态**：由于隐藏了支付功能，属于非经营性网站，只需要 ICP 备案。

---

### 🚨 备案前必须注意的事项

#### 1. 服务器位置要求
- ❌ **国外服务器（如 Vercel、Netlify、Supabase）无法备案**
- ✅ **必须使用中国大陆境内的服务器**
- 推荐服务商：
  - 阿里云
  - 腾讯云
  - 华为云
  - 百度云

#### 2. 域名要求
- 域名必须实名认证
- 域名所有者信息必须与备案主体一致
- 域名后缀必须是工信部批准的（.top 域名可以备案）

#### 3. 主体要求
**个人备案**：
- 中国大陆身份证
- 手机号（实名认证）
- 本人照片（部分省份需要）

**企业备案**：
- 营业执照
- 法人身份证
- 企业公章
- 法人授权书（如非法人办理）

#### 4. 内容合规性
**禁止内容**（会被拒绝或注销）：
- ❌ 涉及政治、宗教、赌博、色情
- ❌ 新闻、出版、药品、医疗
- ❌ 视频、音乐等需要专门许可证的内容
- ❌ 论坛、社区（需要前置审批）
- ❌ 用户生成内容（UGC）平台
- ❌ 即时通讯服务
- ❌ 区块链信息服务（需要单独备案）

**vaid.top 合规建议**：
- ✅ 定位为"技术展示平台"
- ✅ 强调"开发者工具"性质
- ✅ 避免提及"区块链"、"加密货币"
- ✅ 不要暗示任何法律效力
- ✅ 明确说明"不建立法律所有权"

### 📝 备案流程详解

#### 第一步：准备材料
1. **个人备案**：
   - 身份证正反面扫描件
   - 手机号（接收验证码）
   - 邮箱
   - 本人半身照（白色背景，部分省份）

2. **企业备案**：
   - 营业执照副本扫描件
   - 法人身份证正反面扫描件
   - 企业公章电子版
   - 法人手机号和邮箱

#### 第二步：迁移到国内服务器

**重要**：必须先完成服务器迁移才能开始备案。

**推荐方案 1：使用阿里云**
```bash
# 购买阿里云 ECS 或轻量应用服务器
# 最低配置：1核2G，带宽1Mbps，40GB存储
# 预估费用：300-500元/年

# 部署方式：
1. 安装 Node.js 18+
2. 安装 Nginx 作为反向代理
3. 使用 PM2 管理进程
4. 配置 SSL 证书（Let's Encrypt）
```

**推荐方案 2：使用阿里云 OSS + CDN**
```bash
# 静态网站托管方案
# 费用更低，性能更好

1. 构建项目：npm run build
2. 上传 dist 目录到 OSS
3. 配置 CDN 加速
4. 绑定自定义域名
```

**Supabase 数据库问题**：
- Supabase 服务器在国外，无法备案
- **解决方案**：
  - 迁移到阿里云 RDS PostgreSQL
  - 或使用腾讯云数据库
  - 保持 Supabase Edge Functions 用于开发环境

#### 第三步：在服务商申请备案

**以阿里云为例**：
1. 登录阿里云备案系统：https://beian.aliyun.com/
2. 点击"开始备案"
3. 填写主体信息：
   - 省份、城市
   - 证件类型和号码
   - 联系方式
4. 填写网站信息：
   - 网站名称：**V-ID 数字资产展示平台**（不要包含敏感词）
   - 网站内容：**其他**
   - 语言：中文简体
   - 网站描述：面向开发者的数字资产存档与身份识别技术展示平台
5. 上传资料：
   - 真实性核验单（下载模板、打印、签字、扫描）
   - 身份证/营业执照
   - 其他要求的材料

#### 第四步：初审（1-3个工作日）
- 服务商审核材料真实性
- 检查网站内容是否合规
- 可能要求补充材料或修改

#### 第五步：短信核验
- 工信部发送验证码到预留手机号
- 必须在 24 小时内完成验证
- 访问：https://beian.miit.gov.cn/

#### 第六步：管局审核（7-20个工作日）
- 各省通信管理局最终审核
- 不同省份时间不同
- 北京、上海通常较快（7-10天）
- 其他省份可能需要 15-20 天

#### 第七步：备案成功
- 收到工信部短信和邮件通知
- 获得备案号：例如 **京ICP备12345678号**
- 在网站底部添加备案号并链接到工信部网站

### 🚫 常见拒绝原因及应对

#### 1. "网站名称不规范"
**拒绝原因**：
- 包含"中国"、"中华"等词汇
- 暗示官方或权威性质
- 包含地名（除非企业名称）
- 过于宽泛（如"资源网"）

**应对方法**：
- 使用：V-ID 数字资产展示平台
- 避免：V-ID 中国、V-ID 官网

#### 2. "网站内容与实际不符"
**拒绝原因**：
- 网站包含未备案的经营性内容
- 网站功能超出备案范围

**应对方法**：
- 备案前确保隐藏支付功能
- 去除任何商业化元素
- 强调"技术展示"性质

#### 3. "主体信息不一致"
**拒绝原因**：
- 域名所有者与备案主体不同
- 联系方式无法核实

**应对方法**：
- 确保域名实名信息与备案主体一致
- 使用有效的手机号和邮箱

#### 4. "服务器不在可备案范围"
**拒绝原因**：
- 使用境外服务器
- 服务商未接入工信部系统

**应对方法**：
- 必须迁移到国内服务商
- 选择有备案资质的大型云服务

## 🎉 获得备案号后的操作

### 1. 更新网站底部
备案成功后，工信部会发送备案号（如：京ICP备12345678号）

修改代码中的占位文本：

```typescript
// src/App.tsx 第 488 行附近
<a href="https://beian.miit.gov.cn/" target="_blank" rel="noopener noreferrer">
  ICP 备案号: 京ICP备12345678号  // 替换为实际备案号
</a>
```

### 2. 公安备案（部分地区要求）
部分省市还要求进行公安备案：
1. 访问：http://www.beian.gov.cn/
2. 注册并登录
3. 填写网站信息
4. 获得公安备案号
5. 在网站底部添加图标和链接

### 3. 恢复正常运营
```bash
# 关闭维护模式
VITE_MAINTENANCE_MODE=false

# 重新构建
npm run build

# 部署到生产环境
```

### 4. 考虑是否恢复支付功能
如果需要恢复支付功能：
- ❗ 必须申请 **ICP 经营许可证**（ICP 证）
- 流程更复杂，通常需要企业主体
- 审核更严格，需要提供更多材料
- 费用更高（可能需要代办服务）
